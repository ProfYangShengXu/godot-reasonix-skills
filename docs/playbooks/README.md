# Playbook 示例

本目录包含自然语言→工具链的映射示例，供 Agent 在收到用户请求时参考。
每个文件对应一类常见游戏开发需求。

## 索引

| Playbook | 对应场景 | 涉及模块 |
|----------|---------|---------|
| `new-project.md` | 从零创建新项目 | M1, M2, M3 |
| `from-template.md` | 从模板 Fork 项目 | M5, M6 |
| `add-player.md` | 添加玩家角色 | M2, M3 |
| `add-enemy.md` | 添加敌人 | M2, M3 |
| `import-art.md` | 导入美术资源 | M4, M2 |
| `debug-check.md` | 用 Godot CLI 验证项目 | M6 |
